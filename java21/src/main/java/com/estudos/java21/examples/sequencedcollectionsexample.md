# `SequencedCollectionsExample`

Exemplo: `SequencedCollectionsExample.java`.

## Objetivo

**`SequencedCollection`**: ordem explícita com `getFirst`, `getLast`, vista **`reversed()`** sem copiar a lista inteira.

## Nuances

- `reversed()` é vista da sequência — mutações podem refletir conforme a implementação (consulte Javadoc).
- Disponível em listas/deques que implementam a interface.
