# `ChallengeSequencedPipeline`

Solução: `ChallengeSequencedPipeline.java`.

## Objetivo

Iterar com **`reversed()`** em `SequencedCollection` — soma ilustra o percurso invertido (mesmo total numérico que ordem direta para soma simples).

## Nuances

- API nova em coleções ordenadas; contraste com `Collections.reverse` que altera ou copia.
