# `ControlFlow`

**English:** [controlflow-en.md](controlflow-en.md)

Exemplo: `ControlFlow.java`.

## `if` / `else`

Executa um bloco ou outro conforme uma **condição booleana**.

## `for`

Três partes: inicialização; condição de continuação; atualização (ex.: `i++`). Adequado quando sabes quantas voltas ou percorres um intervalo.

## `while`

Repete enquanto a condição for verdadeira; a variável de controlo pode ser atualizada **dentro** do corpo.

## `switch`

No exemplo, sintaxe com **`->`** (Java 14+): cada `case` associa um valor a um bloco ou expressão. O `default` cobre o resto.

## `break` / `continue` (não estão no exemplo)

- **`break`:** sai do `switch` ou do laço mais interior.
- **`continue`:** salta para a próxima iteração do laço.
