# `ControlFlow`

**PT:** [controlflow.md](controlflow.md)

Example: `ControlFlow.java`.

## `if` / `else`

Runs one block or another based on a **boolean condition**.

## `for`

Three parts: initialization; continuation condition; update (e.g. `i++`). Fits when you know how many iterations or traverse a range.

## `while`

Repeats while the condition is true; the control variable may be updated **inside** the body.

## `switch`

In the example, syntax with **`->`** (Java 14+): each `case` maps a value to a block or expression. `default` covers the rest.

## `break` / `continue` (not in the example)

- **`break`:** exits the `switch` or innermost loop.
- **`continue`:** jumps to the next loop iteration.
