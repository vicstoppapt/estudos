# `TextBlocksExample`

Exemplo: `TextBlocksExample.java`.

## Objetivo

Literais **multilinha** com `"""` e remoção de indentação incidental (alinhar conteúdo à esquerda do bloco).

## Nuances

- Escape `\"` raro dentro do bloco; novas linhas reais no código.
- Útil para JSON, SQL, HTML embutido em testes e mensagens.
